# How to Upload These Mermaid Flowcharts to GitHub and Make a QR Code

## Option A: Easiest Method — Create a GitHub Repository

1. Go to GitHub and log in.
2. Click the **+** button in the top-right corner.
3. Click **New repository**.
4. Repository name suggestion: `settle-it-flowcharts`
5. Choose **Public** if you want judges to open it easily by QR.
6. Click **Create repository**.
7. Upload the `README.md` file from this ZIP.
8. Upload the `.mmd` files too, if you want to keep each chart separately.
9. Open the repository main page.
10. GitHub should automatically render the Mermaid diagrams inside the `README.md`.

## Make the QR Code

1. Copy the GitHub repository URL.
2. Open any QR code generator.
3. Paste the GitHub URL.
4. Download the QR code as PNG.
5. Put the QR code on your slide.
6. Add text under it: **Scan to view full Settle It user-flow map**

## Best Slide Usage

On the actual pitch slide, only show this short flow:

Upload → Verify → Confirm → Offer → Fund

Then place the QR code on the side for full detailed flows.
